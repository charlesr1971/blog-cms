Cookie Policy

Cookies from this website, are primarily created to build a secure environment, through which a user can navigate safely, without the loss of any personal data.

The following cookies are set:
  1. User Token: this allows the app to track a user and deliver a personalized experience
  2. Cookie Acceptance: this allows the website to store the 'cookie policy' agreement outcome
  3. Port: this allows the website to store the system port value, which enables the app to pass data from the front end of the website to the server. This is only used for technical development purposes.
  4. Theme: this allows the website to store the current visual styles theme, which is either 'light' or 'dark'

The data stored in a cookie is never passed on to any Third Parties.

When an account is deleted, all cookies are permanently deleted and no user data is persisted in the database.
